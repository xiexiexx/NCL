\section{代码抄录}

可以直接列出代码(设定缩进距离为2, 每行编号并从0开始):
\begin{Verbatim}[tabsize=2,numbers=left,firstnumber=0]
for (int i = 0; i < n; ++i)
	for (int j = 0; j < n; ++j)
    cout << i * j << ' ';
\end{Verbatim}

	也可以使用\texttt{VerbatimInput}命令直接包含代码文件.

	代码变量是\texttt{i}, 复杂的是\Verb@int i = 0@.
